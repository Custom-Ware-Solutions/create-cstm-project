Jak použít:

1) Vytvoř si nové repo na GitHubu
    Jdi na GitHub.
    Klikni na New repository.
    Zadej název, třeba multi-country-social-planner.
    Nedávej tam zatím žádný README / .gitignore / licence (ať je repo prázdné).
    Klikni Create repository.
    Zkopíruj si URL repa (SSH nebo HTTPS), např.:
    git@github.com:tvuj-ucet/multi-country-social-planner.git

2) Vygeneruj projekt ze scaffoldu lokálně
    Otevři terminál.
    V terminálu: cd kam/chceš/projekty (vyber si kde bude root složka projektu)
    Zadej příkaz: npx github:Custom-Ware-Solutions/create-cstm-project multi-country-social-planner (ten druhý parametr je název repozitáře s projektem)
    Proběhne inicializace, vytvoří se složka a je HOTOVO!
